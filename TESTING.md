## Install

<p align="center">
  <a href="https://expo.dev/artifacts/eas/29DyQzPgjq24rXPCTxpKCn.apk">
    Download APK
  </a>
</p>

1. Download the APK on your Android phone
2. Allow "Install from unknown sources" when prompted
3. If upgrading from an older version, uninstall the old one first

**Updates are automatic** — the app checks for patches every launch.

## Wallet Setup

1. Install [Phantom](https://play.google.com/store/apps/details?id=app.phantom) (or use Seed Vault on Seeker)
2. Switch to **Devnet**: Phantom > Settings > Developer Settings > Testnet Mode > Devnet
3. In the app: tap the wallet pill (top of screen) to connect
4. Go to **Me** tab > **Get Test Tokens** for free devnet SOL + USDC
